# **App Name**: StatusKeeper

## Core Features:

- WhatsApp Status Display & Save: View available WhatsApp statuses from local storage and save desired media (images/videos) to the app's internal storage.
- Saved Status Gallery: Access and browse all previously saved WhatsApp statuses within a dedicated gallery.
- Bottom Navigation Bar: Intuitive navigation with 'Status', 'Saved', 'Premium', and 'Settings' tabs.
- App Settings: Configure basic application preferences and manage storage permissions.
- Permissions Management: Handle necessary file system permissions for reading WhatsApp status folders and writing saved media.
- Ad Integration: Display non-intrusive advertisements to support monetization, in line with user's offline-first request.

## Style Guidelines:

- Primary Color: #1F1FCC (deep blue-purple). A strong, modern shade to convey technological efficiency and reliability, offering good contrast with the background.
- Background Color: #ECECF2 (very light blue-purple). A subtly tinted, nearly white background to provide a clean and expansive feel, ensuring readability and a focus on content.
- Accent Color: #3DD8F2 (vibrant cyan). Used for interactive elements, highlights, and calls to action, providing a refreshing and attention-grabbing contrast.
- Headline and Body Font: 'Inter' (sans-serif). Chosen for its neutral, modern, and highly legible characteristics, suitable for a functional and clean interface across all text elements.
- Use a consistent set of simple, line-art icons that clearly represent actions like 'save', 'view', 'settings', and navigation categories.
- Clean, structured layout with clear content hierarchy. Generous white space will enhance readability and provide a modern aesthetic, with a persistent header and bottom navigation bar.
- Incorporate subtle, smooth animations for screen transitions, tab switching, and button presses to create a polished and responsive user experience.
